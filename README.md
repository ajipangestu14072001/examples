Lokio is an Assistant CLI that will help us in making projects that are more structured, fast and efficient.


## Installing Lokio
```bash
# with bun (recommended)
bun add -g lokio@latest  
# with npm
npm i -g lokio@latest  
# with yarn
yarn global add lokio@latest  
# with pnpm
pnpm add -g lokio@latest  
```

## Usage
```bash
lokio
```

## Information

Lokio is a cutting-edge, multi-purpose project scaffolding tool designed to seamlessly integrate with any development environment.

Founded by Wahyu Agus Arifin, Lokio is proudly open-sourced, empowering developers globally to streamline their workflows and unleash their creativity. By simplifying the project setup process, Lokio eliminates the hassle of repetitive boilerplate code, allowing developers to focus on what truly matters: bringing their innovative ideas to life.

With its intuitive design and robust features, Lokio is committed to delivering a reliable, efficient, and developer-friendly experience. Whether you're building a small prototype or a large-scale application, Lokio ensures that your project setup is faster, smoother, and more enjoyable than ever before.
